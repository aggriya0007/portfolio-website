PROJECT_01 / HARDWARE + EMBEDDED

Fire Fighting Robot

A fully self-navigating rescue-bot built from the chassis up on an Arduino Uno. Three flame sensors are mounted around the frame to continuously scan for a fire source; the moment a flame is detected, the onboard logic steers the 4-wheel drive toward it and triggers a relay-controlled water pump to extinguish it — no human operator required.

The control board runs a custom Embedded C++ program that fuses sensor input with motor-driver output in real time, while a dual-motor driver module and 18650 Li-ion cell pack handle the drive train. It was as much an exercise in circuit design and power management as it was in code — every wire on that board is hand-routed and hand-soldered.

PROJECT_02 / COMPUTER VISION + ML

Driver Drowsiness Detection System

A real-time safety system built in Python that watches the road through the driver's eyes — literally. Using OpenCV and MediaPipe's facial landmark model, it tracks 68 points on the driver's face frame-by-frame and computes the Eye Aspect Ratio (EAR) to measure how open or closed the eyes are.

When the EAR drops below a calibrated threshold for a sustained number of frames — the tell-tale sign of eyelids drifting shut — the system fires an audio alert instantly through SoundAudio, giving the driver a critical second to react. It runs entirely on live video, processing frames with NumPy for the numerical work behind the geometry.
